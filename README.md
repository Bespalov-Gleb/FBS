# FBS Order Management System

Web-приложение для управления FBS-заказами с интеграцией маркетплейсов Ozon и Wildberries.

## 📋 Описание проекта

Система предназначена для автоматизации процесса сборки и упаковки заказов с маркетплейсов. Основные возможности:

- 📦 Единый интерфейс для работы с заказами Ozon и Wildberries
- 🔍 Быстрый поиск по артикулам
- 🖨️ Автоматическая печать этикеток и штрихкодов
- 👥 Управление пользователями (администраторы и упаковщики)
- 📊 Отслеживание статусов заказов
- 🔐 Безопасное хранение API ключей маркетплейсов
- ⚡ Real-time синхронизация с маркетплейсами

## 🏗️ Архитектура

Проект построен на современном технологическом стеке:

### Backend
- **Python 3.11+** с **FastAPI** - высокопроизводительный API
- **PostgreSQL 15+** - основная база данных
- **Redis 7+** - кэширование и очереди задач
- **Celery** - фоновые задачи и синхронизация
- **SQLAlchemy 2.0** - ORM с async поддержкой
- **Alembic** - управление миграциями БД

### Frontend (планируется)
- **React 18** с **TypeScript**
- **Material-UI** - UI компоненты
- **Redux Toolkit** - управление состоянием
- **React Router** - навигация

### Инфраструктура
- **Docker** и **Docker Compose** - контейнеризация
- **Nginx** - reverse proxy и load balancing
- **Let's Encrypt** - SSL сертификаты

## 🚀 Быстрый старт

### Предварительные требования

- Docker и Docker Compose
- Git

### Установка и запуск

```bash
# Клонирование репозитория
git clone <repository-url>
cd FBS

# Создание .env файла
cd backend
cp .env.example .env
# Отредактируйте .env файл с вашими настройками

# Запуск всех сервисов через Docker Compose
docker-compose -f docker/docker-compose.dev.yml up -d

# Применение миграций БД
docker-compose -f docker/docker-compose.dev.yml exec api alembic upgrade head

# Создание первого администратора
docker-compose -f docker/docker-compose.dev.yml exec api python scripts/create_admin.py
```

API будет доступно по адресу: http://localhost:8000

Документация API (Swagger): http://localhost:8000/api/docs

Flower (мониторинг Celery): http://localhost:5555

## 🔑 Основные модули

### 1. Модуль аутентификации
- JWT токены (access + refresh)
- RBAC (Role-Based Access Control)
- Безопасное хранение паролей (bcrypt)

### 2. Модуль интеграции с маркетплейсами
- Синхронизация заказов с Ozon и Wildberries
- Получение этикеток FBS
- Обновление статусов заказов
- Работа с КИЗ (для Wildberries)

### 3. Модуль управления заказами
- Единый список заказов из всех маркетплейсов
- Поиск и фильтрация
- Назначение заказов на упаковщиков
- Отслеживание статусов

### 4. Модуль печати
- Генерация штрихкодов товаров
- Получение FBS этикеток
- Печать через браузерный API

### 5. Модуль управления пользователями
- CRUD операции
- Ролевая модель (администратор/упаковщик)
- Аудит действий

## 🔐 Безопасность

- ✅ Хеширование паролей (bcrypt)
- ✅ Шифрование API ключей маркетплейсов (Fernet)
- ✅ JWT токены с коротким временем жизни
- ✅ Rate limiting на API endpoints
- ✅ CORS защита
- ✅ SQL injection защита (параметризованные запросы)
- ✅ XSS защита
- ✅ HTTPS в production

## 📊 Масштабирование

### Горизонтальное масштабирование
- Stateless API серверы (можно запускать N инстансов)
- Балансировщик нагрузки (Nginx)
- Масштабируемые Celery workers

### Производительность
- Connection pooling для БД
- Redis кэширование
- Индексы на часто запрашиваемые поля
- Async operations где возможно

### Рекомендации по инфраструктуре

|----------|-----|-----|------|--------------|

## 🛠️ Разработка

### Backend

```bash
cd backend

# Установка зависимостей
poetry install

# Активация виртуального окружения
poetry shell

# Запуск сервера разработки
uvicorn app.main:app --reload

# Форматирование кода
black app tests
ruff check --fix app tests

# Проверка типов
mypy app
```

### Создание миграций

```bash
# Автогенерация миграции
alembic revision --autogenerate -m "Description"

# Применение миграций
alembic upgrade head

# Откат миграции
alembic downgrade -1
```
