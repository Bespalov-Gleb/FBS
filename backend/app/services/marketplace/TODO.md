## Вопросы для уточнения (ответы из ТЗ)

1. **Нужна ли поддержка нескольких складов одновременно?** — **Да.** ТЗ: «Отображение заказов из всех подключенных складов в едином списке», «Фильтрация по складам», «Цветовое выделение складов (цвет настраивается в разделе Маркетплейсы)».

2. **Какой формат этикеток предпочтителен (PDF, PNG, SVG, ZPL)?** — **PDF (Ozon) и base64/PNG (WB).** ТЗ: «Печать через браузерный Web Printing API», «шаблоны для термопринтеров», «формат этикеток (58мм, 80мм)». Ozon отдаёт PDF, WB — base64 изображения. Для термопринтеров — HTML/CSS шаблоны в браузере.

3. **Нужна ли поддержка batch операций для печати этикеток?** — **Нет (MVP).** ТЗ: «Автоматическая печать двух этикеток при клике на артикул» — печать по одному заказу. Batch не описан в MVP.

4. **Требуется ли offline режим работы?** — **Нет (MVP).** ТЗ: офлайн-режим указан в плане развития 1.1 (после MVP).

5. **Нужна ли синхронизация остатков товаров?** — **Нет (MVP).** ТЗ: «Интеграция с 1С для синхронизации остатков» — в плане 1.2 (версия 1.2). В MVP — только получение данных о товарах и складах.
