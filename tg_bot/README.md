# TG Bot: обработка Excel Ozon/WB

## Что делает

- Принимает несколько Excel-файлов в Telegram.
- Определяет тип файла по первому столбцу:
  - Ozon: `Номер заказа`
  - WB: `№ задания`
- Формирует итоговый файл с двумя листами: `ozon` и `wb`.

## Логика Ozon

- Берутся столбцы `Артикул` и `Количество`.
- Проверяется сумма `Количество` и число строк.
- Если не совпадает (и сумма больше числа строк), строки с количеством `n` разворачиваются в `n` строк.
- После разворачивания значение количества в исходных строках не меняется (в итог выносится только артикул).

## Логика WB

- Берутся столбцы `Размер` и `Артикул продавца`.
- Формируется значение `Артикул_Размер`.

## Итоговый файл

- На листах:
  - `A1`: имя листа (`ozon` или `wb`)
  - `B1`: общее количество артикулов
- Сортировка артикулов: от А до Я (без учета регистра).
- Артикулы, содержащие `white` (без учета регистра), выносятся в отдельный блок.
- Заполнение идет блоками по 67 строк в колонку (строки `2..68`), переполнение переносится в следующую колонку.
- Настройки печати: альбомная, узкие поля, вписать лист на одну страницу.

## Запуск

1. Установить зависимости:
   - `pip install -r requirements.txt`
2. Задать переменную окружения:
   - `TELEGRAM_BOT_TOKEN=<token>`
3. Запустить:
   - `python main.py`

## Запуск в Docker (отдельно)

1. Перейти в папку:
   - `cd tg_bot`
2. Создать `.env` рядом с `docker-compose.yml`:
   - `TELEGRAM_BOT_TOKEN=<token>`
3. Поднять контейнер:
   - `docker compose up -d --build`
4. Логи:
   - `docker compose logs -f tg-bot`
5. Остановка:
   - `docker compose down`

## Управление

- `/start` — инструкция и клавиатура с кнопками
- **Собрать таблицу** — сформировать итоговый Excel из загруженных файлов
- **Очистить файлы** — удалить текущий набор загруженных файлов
