# Настройки печати этикеток

## Где настраивать

**Настройки печати — на сайте fbs-upakovka.ru** (в диспетчере печати).

- **Принтер** — выбор принтера по умолчанию
- **Этикетки Ozon** — размер страницы (58×40 мм), кнопка «Изменить...»
- **Этикетки Wildberries** — размер страницы

Если ШК ФБС Ozon не влезает в одну этикетку — задайте больший размер (например, 80×40) в настройках на сайте.

## API настроек (бэкенд)

```
GET   /api/v1/print-settings  — получить настройки текущего пользователя
PATCH /api/v1/print-settings  — сохранить (default_printer, ozon_labels, wb_labels)
```

Ответ `ozon_labels`: `{ "width_mm": 58, "height_mm": 40 }`  
Ответ `wb_labels`: `{ "width_mm": 58, "height_mm": 40 }`

При запросе этикеток (`/orders/{id}/barcodes-pdf`, `/orders/{id}/label`) параметры `label_width`, `width`, `height` можно не передавать — бэкенд возьмёт значения из настроек пользователя.

## FBS Print Agent

Агент печати (localhost:9199) — только печать. Настроек в агенте нет.  
При печати сайт передаёт `printer` в теле запроса `POST /print`.

## Настройки принтера Windows

Задайте размер бумаги в драйвере принтера: 58×40 мм или 80×40 мм.
